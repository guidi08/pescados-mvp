# Design System (MVP) — estilo “food delivery” moderno

Objetivo: visual limpo, com cards, tipografia forte e CTA claro.

## Tokens

### Cores (sugestão)
- Primária: `#E21A2C` (vermelho)
- Texto: `#111111`
- Subtexto: `#666666`
- Fundo: `#F7F7F7`
- Card: `#FFFFFF`
- Borda: `#EEEEEE`
- Sucesso: `#166534`
- Atenção: `#B45309`
- Erro: `#B91C1C`

### Tipografia
- Fonte: Inter (ou System default)
- Títulos: 22–24, weight 800
- Subtítulo: 16, weight 700
- Corpo: 14–16, weight 400–600

### Componentes
- Card: radius 12, border 1px `#eee`, padding 12–14
- Botão primário: fundo cor primária, texto branco, radius 12
- Badge: radius 999, padding 6x10

## Padrões de tela
- Home: lista de fornecedores (cards)
- Seller: lista de produtos (cards) + search (futuro)
- Product: detalhes + selector de calibre + input quantidade + CTA add to cart
- Cart: lista itens + subtotal + CTA checkout
- Checkout: resumo + métodos de pagamento

## Observação
Evite copiar assets/marca do iFood. Use apenas padrões de UX (cards, tabs, hierarquia).
